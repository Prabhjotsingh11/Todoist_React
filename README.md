Todoist Web App
This is a Todoist web application built using React Native for the frontend, Node.js for the backend, and PostgreSQL as the database via Prisma ORM. The app uses Redux for global state management.

Features
User Authentication: Secure login and signup functionality.
CRUD Operations for Todos: Create, read, update, and delete tasks.
Global State Management: Managed using Redux.
Responsive Design: Optimized for mobile and web platforms.
PostgreSQL Database: Managed using Prisma ORM with NeonDB for data storage.


Tech Stack

Frontend:
React Native
Redux (for state management)
Tailwind CSS (optional for styling)

Backend:
Node.js
Express.js
Prisma (ORM)

Database:
PostgreSQL (via NeonDB)
